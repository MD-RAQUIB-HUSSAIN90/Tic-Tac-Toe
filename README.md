# 🎮 Tic Tac Toe Game

A modern and interactive **Tic Tac Toe** game built using **HTML**, **CSS**, and **Vanilla JavaScript**.

This project provides a smooth gaming experience with responsive design, score tracking, sound effects, winner detection, draw detection, and modern UI animations.

---

## 🚀 Features

- 🎮 Classic Tic Tac Toe Gameplay
- 🏆 Winner Detection
- 🤝 Draw Detection
- 📊 Live Score Board
- 🔄 Reset Game
- 🆕 New Game
- 🔊 Sound Effects
- 🌙 Dark / Light Mode
- 🎉 Winner Animation
- 📱 Fully Responsive Design
- ⚡ Optimized JavaScript
- 💻 Clean and Modern UI

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
